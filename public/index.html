Your job is to implement the data structures for a simple tracking application for keeping a log of work tasks. 

Create an object in JS called task. Setup the properties in this object to meet the needs outlined in the data model specifications above. Here is a hint to get you started:

let task = {
  name: "Initial Sketches",
  ...
}

For the id property, for now just use the following value:

Date.now()

"The Date.now() static method returns the number of milliseconds elapsed since the epoch, which is defined as the midnight at the beginning of January 1, 1970, UTC." (MDN Web Docs) This provides a suitable enough value for our purposes to use as a unique id.

For the date property, we're going to also use the JavaScript Date object, with some additional formatting code so it is readable as a date. You can use the following value:

new Date().toISOString()

Note: The ISO 8601 Date Standard is very useful for storing dates as it represents date and time values numerically in an internationally agreed format: year, month, day, hour, minutes, seconds, and milliseconds. For example, May 1, 2023 at 6:30 p.m. is represented as 2023-05-01 18:30:00.000. Due to the descending order of date and time units, these values can also be chronologically sorted very easily!

Try to log the task object out to the console to see what it looks like.

console.log(task)

Create an empty array called taskList. Then write a function called addTask() which has all the required input parameters to create a task object (i.e. all the properties from the task object that aren't automatically generated):

  • name
  • type
  • rate
  • time
  • client

Cut and paste the task object definition into the function you've just created. In your object definition, replace the static values for the above properties with the input parameters you just added to the function.

At the bottom of the function, add the task object to the taskList array (try to remember from the previous exercise how to add items to arrays).

Call the function with hard-coded input parameters, for example:

addTask("Initial Sketches", "Concept Ideation", 50, 5, "Google");

Come up with 4 more tasks of your own, adding them to the array using the above function call syntax.

Log the array out to the console, showing that it works for all your test tasks. Next week we will look at how to populate these parameters with user input.

console.log(taskList);
